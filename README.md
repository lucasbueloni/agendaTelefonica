📒 Contact Book CLI

A command-line application (CLI) built with Python for contact management.

This project implements the core CRUD (Create, Read, Update, Delete) operations, along with features for marking and viewing favorite contacts. It was developed to practice Python fundamentals, including functions, lists, dictionaries, conditional statements, loops, and code organization.

✨ Features

* Add new contacts
* View all registered contacts
* Edit contact information
* Delete contacts
* Mark contacts as favorites
* View favorite contacts

🛠️ Technologies

* Python 3

📚 Concepts Practiced

* Functions
* Lists
* Dictionaries
* Conditional statements (if, elif, else)
* Loops (for and while)
* Data manipulation
* Code modularization

🎯 Purpose

This project is part of my Python learning journey and was created to strengthen my understanding of core programming concepts, data structures, and command-line application development.
